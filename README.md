# angular-materials

飞冰（ICE）官方 Angular 物料（模板/区块/组件）。

## 使用

对应的物料数据：`https://ice.alicdn.com/assets/angular-materials.json`，可以通过 Iceworks 的自定义物料功能使用。

## 如何贡献

- Fork 仓库，修改代码
- 提交 PR

## 开发指南

### 新增物料

```bash
$ cd angular-materials
$ idev add
```

### 开发调试

```bash
$ cd blocks/Example
$ npm install
$ npm run start
$ npm run build
```
